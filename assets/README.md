# Asset Notes

이 폴더의 PNG 에셋은 과제 기간 중 프로젝트용으로 직접 생성한 절차적 게임 아트입니다.

- `sprites/player_units.png`: 아군 유닛 14종, 4프레임 스프라이트 시트
- `sprites/enemy_units.png`: 적 유닛 17종, 4프레임 스프라이트 시트
- `backgrounds/stage_00_space.png` ~ `stage_09_space.png`: 행성별 우주 배경 레이어
- `vfx/combat_vfx_atlas.png`: 글로우, 슬래시, 충격파, 먼지 VFX 아틀라스
- `ui/pawline_ui_atlas.png`: 버튼/패널 질감 UI 아틀라스
- `cutins/solar_gatekeeper_cutin.png`: 보스 등장 컷인 배너

폰트:

- `fonts/Galmuri11.ttf`: Galmuri 폰트
- `fonts/Galmuri-OFL.md`: Galmuri의 SIL Open Font License 1.1 전문

Galmuri는 SIL Open Font License 1.1로 배포되는 무료 폰트이며, Reserved Font Name은 `Galmuri`입니다.
