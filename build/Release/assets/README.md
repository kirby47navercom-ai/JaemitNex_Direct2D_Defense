# Asset Notes

이 폴더의 PNG 에셋은 과제 기간 중 프로젝트용으로 직접 생성한 절차적 게임 아트와 CC0 무료 외부 에셋을 함께 사용합니다.

- 유닛과 적은 코드 기반 무기/이펙트 연출 위에 GameArt2D Cat and Dog Free Sprites의 캐릭터 프레임을 얹어 사용
- `sprites/gameart2d_catdog/cat_motion_sheet.png`, `dog_motion_sheet.png`: GameArt2D 무료 Cat/Dog 프레임을 게임용 10열 x 7행 시트로 재구성한 캐릭터 모션 시트
- `backgrounds/stage_00_space.png` ~ `stage_09_space.png`: 행성별 우주 배경 레이어
- `vfx/combat_vfx_atlas.png`: 글로우, 슬래시, 충격파, 먼지 VFX 아틀라스
- `vfx/slash_effect_sheet.png`, `vfx/slash_gameboy_effect_sheet.png`: 외부 VFX 이미지를 적용한 타격 슬래시 이펙트
- `vfx/heal_effect_sheet.png`, `vfx/heal_gameboy_effect_sheet.png`: 외부 VFX 이미지를 적용한 월렛/회복 반짝임 이펙트
- `vfx/fire_explosion_sheet.png`, `ice_burst_sheet.png`, `thunder_strike_sheet.png`, `water_impact_sheet.png`: 압축 에셋에서 선별한 속성별 전투 이펙트
- `vfx/dark_impact_sheet.png`, `acid_impact_sheet.png`, `earth_impact_sheet.png`, `smoke_puff_sheet.png`, `holy_flash_sheet.png`: 적/보스/죽음/충돌 연출용 외부 이펙트
- `ui/pawline_ui_atlas.png`: 버튼/패널 질감 UI 아틀라스
- `cutins/solar_gatekeeper_cutin.png`: 보스 등장 컷인 배너

외부 캐릭터 에셋:

- `sprites/gameart2d_catdog/*`: GameArt2D Cat and Dog Free Sprites
- 출처: https://www.gameart2d.com/cat-and-dog-free-sprites.html
- 라이선스: GameArt2D Freebies 라이선스 페이지 기준 CC0

폰트:

- `fonts/Galmuri11.ttf`: Galmuri 폰트
- `fonts/Galmuri-OFL.md`: Galmuri의 SIL Open Font License 1.1 전문

Galmuri는 SIL Open Font License 1.1로 배포되는 무료 폰트이며, Reserved Font Name은 `Galmuri`입니다.
