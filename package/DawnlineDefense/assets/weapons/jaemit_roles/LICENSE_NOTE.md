# Jaemit Role Weapons

이 폴더의 무기 PNG는 프로젝트 안에서 역할별 전투 표현을 맞추기 위해 직접 생성한 투명 배경 스프라이트입니다.

- 사용 목적: 근접 유닛은 검, 방패, 단검, 망치, 창, 드릴처럼 근접성이 읽히는 무기를 사용합니다.
- 사용 목적: 원거리 유닛은 스태프, 포커스, 캐논처럼 발사 지점과 투사체가 자연스럽게 이어지는 무기를 사용합니다.
- 제작 방식: PowerShell과 System.Drawing으로 단순 도형, 색상 팔레트, 외곽선, 글로우를 조합해 생성했습니다.
- 외부 유료/저작권 에셋을 포함하지 않습니다.
