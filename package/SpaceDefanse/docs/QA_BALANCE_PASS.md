# QA / 밸런스 패스 기록

## 목적

제출 전 빌드에서 평가자가 바로 확인할 수 있는 흐름을 안정화하기 위해 전투, UI, 시연, 옵션 안전장치를 함께 점검했다.

## 이번 패스에서 추가한 항목

- 타이틀 화면에 `시연 모드` 버튼을 추가했다.
- 타이틀 `D` 키로 자동 시연 전투를 바로 시작할 수 있게 했다.
- 옵션 화면에 `눈부심 줄이기`를 추가해 전체 화면 플래시를 낮출 수 있게 했다.
- 옵션 화면에 `진행 데이터 초기화`를 추가하고, 실수 방지를 위해 두 번 확인해야 초기화되도록 했다.
- 브리핑 화면에 스테이지 위협도와 편성 전력을 비교하는 밸런스 패널을 추가했다.
- 전투 초반 튜토리얼을 단계형 안내로 바꿔 카드, 월렛, 카메라, 캐논을 순서대로 보여준다.
- 태양 스테이지 최종 클리어 시 `CAMPAIGN CLEAR` 결과 연출을 추가했다.
- AI 디렉터가 전선 우세/열세에 따라 적 스폰 압력을 미세 조정하도록 했다.

## 수동 QA 체크리스트

- 타이틀에서 `시작하기`, `시연 모드`, `옵션`, `나가기` 버튼 위치가 겹치지 않는지 확인한다.
- 옵션에서 `피격 흔들림`, `눈부심 줄이기`, 속도, 화면 안전 여백, 진행 초기화 버튼이 모두 클릭되는지 확인한다.
- 옵션에서 `X` 또는 초기화 버튼을 한 번 눌렀을 때 바로 초기화되지 않고, 두 번째 확인에서만 초기화되는지 확인한다.
- 브리핑 화면에서 `BALANCE` 패널이 행성 정보, 편성 카드, 난이도 버튼과 겹치지 않는지 확인한다.
- 전투 시작 18초 안에 튜토리얼 안내가 카드/월렛/카메라/캐논 순서로 표시되는지 확인한다.
- `D` 또는 타이틀 `시연 모드`로 자동 전투가 시작되고, 유닛 소환/월렛/캐논/카메라 이동이 보이는지 확인한다.
- `F1`로 전투 중 시연 모드를 켜고 끌 수 있는지 확인한다.
- 태양 스테이지 클리어 시 일반 `STAGE CLEAR`가 아니라 `CAMPAIGN CLEAR`가 표시되는지 확인한다.
- `눈부심 줄이기`가 켜진 상태에서 캐논 사용 시 전체 화면 밝기 변화가 줄어드는지 확인한다.

## 밸런스 판단 기준

- 스테이지 위협도는 적 기지 체력, 적 등장 간격, 스테이지 위협 배율, 보스 첫 등장 시간을 함께 본다.
- 편성 전력은 유닛 체력, 공격력, 사거리, 이동 속도, 공격 주기, 비용, 레벨, 시너지를 함께 본다.
- 편성 전력이 낮으면 `상점 강화 추천`, 비슷하면 `보통 난이도 적정`, 충분히 높으면 `어려움 도전 가능`을 표시한다.
- AI 디렉터는 플레이어가 우세할 때만 적 압력을 조금 올리고, 플레이어 기지가 위험하면 압력을 낮춘다.

## 3분 시연 추천 흐름

1. 타이틀에서 `시연 모드` 버튼을 누른다.
2. 자동 소환, 월렛 강화, 카메라 이동, 캐논 사용을 40~60초 정도 보여준다.
3. `Esc` 메뉴를 열어 언제든 나갈 수 있는 옵션을 보여준다.
4. 메뉴로 돌아가 상점과 브리핑의 `BALANCE` 패널을 짧게 보여준다.
5. 테스트 모드가 필요하면 `F2` 후 `F5`로 결과 화면과 캠페인 클리어 연출을 확인한다.
