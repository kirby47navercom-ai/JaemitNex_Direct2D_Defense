## 사람이 판단한 부분

- 과제 주제에 맞춰 탑뷰 디펜스 장르로 범위를 제한
- 직접 만든 PNG 게임 에셋을 중심으로 구성하고, 폰트만 OFL 라이선스가 확인된 Galmuri를 포함하는 방향 선택
- 제출물 구조를 실행 파일, 문서, zip으로 구성
- 유닛 종류, 적 종류, 행성 스테이지 구성, 지갑 성장, 캐논, 조작 방식 선택
- 시작 화면, 옵션 화면, 상점 화면, ESC 메뉴, 결과 화면, 캐릭터 편성 화면의 구성 선택

## 제출 리포트에 넣을 수 있는 요약

이번 작업에서는 AI를 단순 코드 생성기가 아니라 게임 제작 보조자로 활용했습니다. 먼저 과제 요구사항을 기반으로 실행 가능한 범위를 정하고, AI의 도움을 받아 Direct2D 기반 프로젝트 구조와 핵심 게임 루프를 빠르게 구성했습니다. 이후 유닛 카드 소환, 지갑 업그레이드, 적 자동 스폰, 유닛 전투, 투사체 충돌, 기지 파괴, 시작/옵션/상점/ESC/브리핑/결과/도감 UI, 행성 스테이지 선택과 순차 잠금, 행성별 배경과 실제 전투 기믹, 위험 범위 예고, 행성별 보스 패턴/2페이즈, 14종 캐릭터 편성, 한글 캐릭터명, 유닛 진화, 편성 시너지, 난이도 선택, 테스트 모드, 데모 자동 전투, 브리핑 밸런스 판정, AI 디렉터, 스테이지별 적 조합, 스테이지 클리어 보상, 유닛 구매와 강화, 성장 추천, 진행 저장, 게임 속도 조절, 화면 안전 여백 조절, 눈부심 줄이기, 진행 데이터 초기화 확인, 캐릭터별 무기/공격 이펙트, 상태머신 기반 애니메이션, 타입별 파티클, 보스 컷인과 카메라 반응, 태양 최종 클리어 연출, 델타타임 기반 업데이트 구조를 검토하며 하나의 플레이 가능한 결과물로 통합했습니다. 또한 HLSL 셰이더 레퍼런스를 작성하고, 제출 안정성을 위해 같은 의도의 후처리 효과와 카툰풍 하프톤/액션 라인/5x7 픽셀 HUD, 우주 배경, 로컬 조명과 방향성 그림자를 Direct2D 절차적 VFX 패스로 구현했습니다.
